The Cheat Sheet

I know this isn't very long, but it doesn't have to be. I plan to continually update as time goes on.

Command Description Example 
ls
(list)		 Lists the contents of the current directory. 
ls <enter>
 
cd
(change directory)		 Moves to another directory. 
cd ~/Desktop <enter>
 
wc
(word count)		 Counts the number of words it is given.
This example counts the items in a directory.		 
ls -1 | wc -l <enter>
