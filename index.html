The Linux Command Cheat Sheet

Command Description Example 
ls
(list)		 Lists the contents of the current directory. 
ls <enter>
 
cd
(change directory)		 Moves to another directory. 
cd ~/Desktop <enter>
 
wc
(word count)		 Counts the number of words it is given.
This example counts the items in a directory.		 
ls -1 | wc -l <enter>
 
sudo
(superUserDo)		 Acts as a modifier for other commands to temporarily give them root permissions. Do not use this unless you know exactly what the command you are using is going to do.
This example edits an SSH configuration file.		 
sudo vim sshd-config <enter>
 
cat
(concatenate)		 Concatenates files together or acts as a quick way to spit to contents of a file out into the terminal. 
cat style.css <enter>
